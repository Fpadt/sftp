# sftp
R package with convenience functions for SFTP operations wrapping RCurl functions

* sftp_connect
* sftp_list
* sftp_listfiles
* sftp_listdirs

* sftp_download
* sftp_upload
* sftp_delete
* sftp_rename

* sftp_makedir
* sftp_removedir
* sftp_changedir

### Motto
_Cool data science is sexy, but it is also nice to get hold of the data to begin with_
